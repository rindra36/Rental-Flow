# **App Name**: RentalFlow

## Core Features:

- Add Apartment: Allows the user to add a new apartment to the system with its name and monthly rent.
- Edit Apartment: Enables the user to modify existing apartment details, including name and rent amount.
- Delete Apartment: Functionality to remove an apartment from the system, including associated leases and payments.
- Manage Leases: Provides an interface to view, add, edit, and delete leases for each apartment.
- Record Payment: Allows users to record payments against specific leases, including the amount, date, and full payment status.
- Financial Overview: Presents a summary of expected income, collected payments, and outstanding balances for a selected month, and the occupancy
- Data Backup/Restore: The app should remind the user to manually trigger periodic backups of their MongoDB Atlas data to external storage (user's local drive, Google Drive, etc.) using provided MongoDB tool hints. The backed up files could then be used to restore data from within the app, if desired. There are no features for automated backups.

## Style Guidelines:

- Primary color: Indigo (#6366F1) to convey trust and professionalism.
- Background color: Light gray (#F9FAFB) for a clean and modern feel.
- Accent color: Teal (#2DD4BF) for interactive elements and highlights.
- Body font: 'Inter', sans-serif, for clear and readable text. This is also the header font.
- Use consistent and clear icons from Lucide React for actions and navigation.
- Maintain a clean and organized layout with sufficient whitespace to improve readability.
- Subtle animations for transitions and feedback to enhance user experience.
